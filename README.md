# Poison Machine
A vulnerable Linux machine built from scratch. Built on top of a vulnerable an old Ubutnu version.
## Vulnerabilities
- Local File Inclusion (LFI)
- SSH Log Poisoning
- A Kernel Exploit (2017-16995)
